# AzaleaCore

### Maven Import

In **pom.xml**:

```xml

  <repositories>
    <!-- repos -->
    <repository>
        <id>azalea-core-repo</id>
        <url>https://github.com/AzaleaLibrary/AzaleaCore/raw/public/</url>
    </repository>
  </repositories>
  
  <dependencies>
    <!-- deps -->
    <dependency>
        <groupId>AzaleaLibrary</groupId>
        <artifactId>AzaleaCore</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>
  </dependencies>

```
