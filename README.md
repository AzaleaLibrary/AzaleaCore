# AzaleaCore

### How to

#### Import `AzaleaCore`

In **pom.xml**:

```xml
  <dependencies>
    <!-- other dependencies -->
    <dependency>
        <groupId>AzaleaLibrary</groupId>
        <artifactId>azalea-core</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>
  </dependencies>
```

> See a working example [here](https://github.com/AzaleaLibrary/ExampleMinigame).
