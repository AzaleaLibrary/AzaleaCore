# AzaleaCore

### How to

#### Import `AzaleaCore`

In **pom.xml**:

```xml
  <dependencies>
    <!-- other dependencies -->
    <dependency>
        <groupId>azalealibrary</groupId>
        <artifactId>azaleacore</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>
  </dependencies>
```

#### Register a minigame

> See a working example [here](https://github.com/AzaleaLibrary/ExampleMinigame).
